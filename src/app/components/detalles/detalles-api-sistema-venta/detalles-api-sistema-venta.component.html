Backend – API RESTful para Sistema de Ventas


Desarrollé una API RESTful utilizando arquitectura en capas (DAL, BLL, DTO, IOC), permitiendo una separación clara de responsabilidades y escalabilidad. Implementé Swagger para facilitar la documentación y pruebas de los endpoints.

Módulos principales:

  • Autenticación y roles: Inicio de sesión, control de acceso por roles.
  • Gestión de usuarios, productos y categorías: CRUD completo.
  • Ventas: Registro, historial y reportes filtrables.
  • Dashboard: Cálculo dinámico de métricas semanales.
  • Menu dinámico: Retorna funciones visibles según el usuario autenticado.

    • Gracias a este proyecto, consolidé mi conocimiento en programación orientada a objetos, inyección de dependencias, LINQ, expresiones lambda y depuración. Aprendí a desarrollar APIs organizadas, eficientes y seguras, listas para ser consumidas por el frontend.


    Tecnologías: C#, .NET, ASP.NET Core, Entity Framework, SQL Server
