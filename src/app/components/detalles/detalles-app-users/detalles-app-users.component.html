Página web tipo plantilla para evento académico


Proyecto realizado como prototipo para la "7ma Semana de Mecatrónica y 1ra Semana de la Ingeniería" de la UPIIZ – IPN (Unidad Profesional Interdisciplinaria de Ingeniería Campus Zacatecas - Instituto Politécnico Nacional).

El objetivo fue presentar al cliente una versión preliminar de lo que podría ser el sitio oficial del evento, permitiéndole visualizar la estructura general y el diseño.
La aplicación fue desarrollada con Angular como SPA (Single Page Application), incorporando Angular Material para lograr una interfaz limpia, moderna y responsiva, manteniendo una estética coherente con la identidad visual de ediciones anteriores del evento.
El contenido visual y textual fue planteado como referencia, sirviendo como guía para el contenido final. Esta plantilla ayudó al cliente a tomar la decisión de avanzar hacia un sistema más robusto, incluyendo:

  • Un portal para preregistro de asistentes y registro a actividades académicas, deportivas y recreativas.
  • Una futura aplicación complementaria para el equipo organizador, enfocada en el manejo de registros, asistencias, altas y bajas.

    • Durante el desarrollo, adquirí experiencia en la construcción de interfaces responsivas con Angular y Material UI, en la personalización de paletas de colores, animaciones, y en la disposición eficiente de elementos para mejorar la experiencia del usuario.


    Tecnologías: JavaScript, HTML/CSS, Angular, Material UI
