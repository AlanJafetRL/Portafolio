Fullstack - Aplicación web con sistema de registro e información para evento académico


Actualmente el proyecto continúa en evolución, con nuevas funcionalidades en desarrollo como control de asistencia, edición de registros y panel para organizadores.

Aplicación web desarrollada con Angular y .NET para la gestión y divulgación de un evento universitario.
Permite a los participantes registrarse en concursos y consultar información clave del evento (fechas, categorías, lugar, etc.) desde una plataforma responsiva y accesible.
El backend implementa una API RESTful conectada a base de datos, mientras que el frontend ofrece formularios validados, diseño visual limpio y estructura adaptable.
El sistema fue desplegado en Azure como versión funcional para presentación al cliente.

Este proyecto me permitió afianzar mis conocimientos en comunicación entre frontend y backend, validación de formularios, despliegue en la nube y diseño de soluciones orientadas al usuario final.


Tecnologías: JavaScript, HTML/CSS, Angular, Material UI, C#, .NET, ASP.NET Core, Entity Framework, SQL Server, Microsoft Azure
