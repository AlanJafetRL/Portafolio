Frontend – Aplicación Web para Sistema de Ventas


Desarrollé el frontend de una aplicación web para la gestión de ventas, con enfoque en experiencia de usuario y arquitectura modular basada en NgModules.
La interfaz se adapta dinámicamente al rol del usuario (Administrador, Empleado, Supervisor), ofreciendo funcionalidades personalizadas.

Principales funcionalidades:

  • Dashboard: Resumen de ventas semanales (ingresos, ventas, productos vendidos).
  • Gestión de usuarios y productos: CRUD completo con tablas interactivas.
  • Registro de ventas: Búsqueda rápida de productos, selección de cantidad, tipo de pago, y resumen final.
  • Historial de ventas: Filtros por fecha o ID.
  • Reportes: Generación y exportación de reportes en Excel.

    • Este proyecto marcó mi primer acercamiento profundo a Angular y Angular Material. Aprendí sobre diseño de componentes, consumo de API RESTful, personalización de interfaces y buenas prácticas en desarrollo frontend.


    Tecnologías: JavaScript, TypeScript, HTML/CSS, Angular, Material UI
